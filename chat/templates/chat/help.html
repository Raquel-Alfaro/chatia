CHAT IA

CHAT IA es una aplicación web, elaborada con Django, que permite crear conversaciones con una inteligencia artificial.

El usuario puede:

  • Iniciar sesion
  • Crear conversaciones
  • Enviar mensajes y recibir respuestas en tiempo real
  • Cambiar el nombre a las conversaciones
  • Eliminar conversaciones
  • Personalizar su perfil

Proyecto realizado con Python, Django, HTML, Bootstrap, OpenRouer API y SQLite
